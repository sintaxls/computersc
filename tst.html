9. What is a semantic error?

  • An error that occurs when the syntax of a program is incorrect, preventing it from compiling.
  • A logical error that occurs when the code runs but produces incorrect or unintended results
  • An error that arises from a typo or misspelled variable name
  • A runtime error that occurs when the program attempts to access invalid memory locations

10. What is a runtime error?

  • An error that occurs while the program is executing, often due to resource issues
  • An error during the compilation process, preventing the program from being created
  • An error caused by incorrect syntax, which is caught before the program runs
  • An error that occurs during the design phase and affects its overall logic
